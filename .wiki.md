# Project Summary
Proxiatech is a web development platform that delivers modern, responsive, and high-performance digital solutions for businesses. It effectively showcases the company's services, portfolio, and client testimonials, establishing Proxiatech as a reliable partner in digital transformation. The platform enhances user engagement and simplifies the user experience with polished professional imagery and immersive interactions. Recent updates include a dark mode feature and the removal of the interactive particle background in the Hero section, further enhancing user experience and accessibility.

# Project Module Description
- **Header**: Modern navigation bar with a white background, enhanced logo, and prominent CTA button. Features a progress bar indicating page scroll progress with a blue to cyan gradient and a dark mode toggle.
- **Hero**: Displays a professional image representing digitalization, now without the interactive particle background.
- **Services**: Overview of services including web and mobile development, IT consulting, and custom software solutions, enhanced with animations.
- **About**: Information about the company's mission, values, expertise, and mastered technologies.
- **Portfolio**: Showcase of completed projects with animated elements.
- **Testimonials**: Client feedback and ratings presented with animations.
- **Contact**: Inquiry form and contact information, featuring smooth animations.
- **Footer**: Updated with a gradient background matching the contact info card for visual coherence.
- **Scroll to Top Button**: A floating button appears after scrolling down 400px, allowing users to smoothly return to the top of the page with a blue to cyan gradient, pulse animation, and hover effects.
- **Loading Screen**: Elegant loading screen with animated Proxiatech logo and blue-cyan gradient, featuring a loading progress bar and dynamic text.
- **Custom Cursor**: Elegant custom cursor that follows the mouse with a trailing effect and enlarges on hover over buttons and links, featuring a blue-cyan gradient.
- **Smooth Transitions**: Added smooth transitions between sections with parallax effects and progressive reveal for an immersive experience.
- **Dark Mode**: New feature allowing users to toggle between light and dark themes across the application.

# Technology Stack
- **Frontend**: React, TypeScript, Vite, Tailwind CSS, Framer Motion.
- **Backend**: Not applicable for this showcase site.
- **Deployment**: Static site hosting.

# Usage
1. Install dependencies:
   ```bash
   pnpm install
   ```
2. Build the project:
   ```bash
   pnpm run build
   ```
3. Lint the code:
   ```bash
   pnpm run lint
   ```
4. Run the application:
   ```bash
   pnpm run dev
   ```
